**PlayVault** is a full-stack video game collection and marketplace platform that allows users to discover games, build personal libraries, and trade or sell items with other users.

The application is built with **Next.js**, **Node.js**, and **PostgreSQL**, and integrates a public video game API to provide rich game metadata for browsing and search.

PlayVault is designed as a real-world product-style application, focusing on clean architecture, scalability, and modern full-stack development practices.
